Given an array of positive numbers and a positive number 'k'
find the maximum sum of any contiguous subarray of size 'k'.


Example 1:

Input: [2, 1, 5, 1, 3, 2], k=3
Output: 9
Subarray with maximum sum: [5, 1, 3].

Example 2:

Input: [2, 3, 4, 1, 5], k=2
Output: 7
Subarray with maximum sum: [3, 4].
